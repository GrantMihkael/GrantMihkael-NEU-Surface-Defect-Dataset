# Manufacturing Defect Detection using CNN and Grad-CAM

## Project Overview
This project develops a CNN-based system for classifying steel surface defects using the NEU Surface Defect Database. The system predicts one of six defect classes and generates Grad-CAM visualizations to highlight image regions that influenced the prediction.

## Dataset
NEU Surface Defect Database  
Selected source: Kaggle mirror of the dataset.

## Defect Classes
- Crazing
- Inclusion
- Patches
- Pitted Surface
- Rolled-in Scale
- Scratches

## Planned Model
- CNN classifier (ResNet18 or MobileNetV2)
- Grad-CAM visualization

## Evaluation Metrics
- Accuracy
- F1 Score
- Confusion Matrix
- False Alarm Rate
